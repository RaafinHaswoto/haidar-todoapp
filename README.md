# Todo App by Impact Byte

A simple todo app to save and list various kinds of things to do.

![mindblown](./images/mindblown.gif)

This is a markdown documentation with `.md` format.

![Screenshot](./images/todo.png)

## Features

In the app, we can:

- Create todo
- List todo
- Delete todo
- Favorite todos
  - Use a star symbol
    - With yellow color
